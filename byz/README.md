# Quick & dirty how to

* Use with 70/64 mm acrylic tube
* Bottom fits a standard 4 AA battery holder and an arduino pro mini
* Cavities for the LEDs fit single pieces of WS2812b strips

### Assembly

1. Wire everything together using enameled copper wire
2. fit acrylic tube into the grooves of the 3D printed parts
3. Glue
4. Send it through Seidenstrasse
5. See it break
